#  Описание
- Базовый проект, включающий все модули и приложения.
- Используется как основа для разработки любого приложения.
- Родительский проект [mp](https://github.com/mostali/udav/tree/master/mp/pom.xml) используется для настройки сборок (профилей)

🌐 Включает  
- Базовый модуль Java Core Utility - https://github.com/mostali/udav_core
- Модуль для работы с БД - https://github.com/mostali/udav/tree/master/utl-db
- Модуль для работы с сетью + html - https://github.com/mostali/udav/tree/master/net 
- Модуль для работы с Telegram Bot Api - https://github.com/mostali/udav/tree/master/nett
- Модуль для работы с VK Bot Api - https://github.com/mostali/udav/tree/master/netv5
- Модуль для работы с Instagram
- Модуль для работы с Google Sheet Api - https://github.com/mostali/udav/tree/master/utl-gd
- Модуль для работы с Qiwi Api 
- Модуль для работы с Selenium 
- Модули для работы с ssh, git, opencv, doc/excel/pdf, xtable, etc
- Модули для работы с Jira=https://github.com/mostali/udav/tree/master/utl-jira
- Модули для работы с Jira & Confluence
- Web-приложение (Демо  http://zznote.ru) - Конструктор сайтов - Заметки - https://github.com/mostali/zznote
