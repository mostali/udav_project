
## Общее описание системы
Сайт - это набор страниц с поддоменамами
Страница - набор форм, из которой состоит страница
Форма - это файл, хранящийся в директории .forms
Форма может содержать свойста - файл .props
При создании формы - к форме дополнительно, всегда создаётся компонент ссылки с именем формы ( файл свойств в директории .coms )

## Пользователи бывают 5-типов
Пользователи бывают 5 типов
1) Owner (полные права)
2) Admin (почти полные права)
3) Editor
   -- держатель поддомена
   -- может редактировать страницы только своего поддомена
   -- не может редактировать страницы других поддоменов (если нет явных прав)
   -- может просматривать/редактировать/запускать другие страницы/формы если Editor выдал разрешения для страницы/формы)
  
4) Anonim (может все что не запрещено системой)

**Plane Permissions**  
{ "user":"loginAlias", "secv":"", "sece":"@", "secr":"@" }


**Page Permissions**  
{ "user":"loginAlias", "secv":"@", "sece":"@", "secr":"@" }

**Form Permissions**  
{ "user":"userlogin", "secv":"@", "sece":"@", "secr":"@" }

**Возможные значения**
@ - разрешено всем  
@@ - разрешено всем зарегестрированным пользователям
"userlogin1,userlogin2" - разрешено только указанным пользователям
