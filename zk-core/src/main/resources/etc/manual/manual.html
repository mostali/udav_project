Общая информация

Java вэб-приложение для создания, хранения, отображения различных данных, создания вэб-страниц

Позволяет создавать web-страницы добавляя разные типы заметок с текстом, html, фото, видео/аудио и др

Заметки могут выполнять более специфичные операции, типа отправки http-запроса, сообщений в кафку, sql-запросов, выполнять задания(quartz), выполнять скрипты, etc

Приложение может работать с заметками как rest-server

Поддерживает работу других пользователей, которые могут получить свой поддомен типа t777.site.com

Типичные операции работы с заметками продублированы в тг бота(Experimental)

Для получения токена пользователям необходимо отправить '+' бот в ВК, или в ТГ

Работа с заметками

- Заметка это блок данных, текст, который хранится в файле

- Заметка в зависимости от содержимого может использоваться как для хранения, отображения, или исполнения

- Если это заметка типа отправки сообщения или выполнения скрипта, то заметка представляет из себя три части

Строка 1 Команда/Url

Строка 2..N Заголовок

Строка N.. Тело сообщения

Поддерживаемые коментарии

Заголовки в зависимости от типа заметки могут поддерживать следующие комментарии

#    Общий коментарий

--#  Заметки типа http, kafka и др.

//     для groovy и др.
